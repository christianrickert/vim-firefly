		  Firefly color scheme    by Christian

Firefly, a color scheme with a unique touch.

==============================================================================
Color scheme options					firefly-options*

						*g:dark_transp_bg*
Set to 1 if you want a transparent background. Takes effect only in the
terminal.
>
	let g:dark_transp_bg = 0
<
